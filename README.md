# 🌍 AI Translator App (Qwen2-0.5B)

A simple web app that translates text using a **local AI model (no API keys required)**.
Built with **Flask + Hugging Face Transformers** using `Qwen2-0.5B-Instruct`.

---

## 🚀 Demo

Example translations:

| Input            | Output             |
| ---------------- | ------------------ |
| I like soccer    | Me gusta el fútbol |
| How are you?     | ¿Cómo estás?       |
| What time is it? | ¿Qué hora es?      |

---

## 🧠 Model

This app uses:

* `Qwen/Qwen2-0.5B-Instruct`
* Runs **locally** (no external APIs)
* Lightweight and efficient for small tasks like translation

---

## 🛠️ Tech Stack

* Python 🐍
* Flask 🌐
* Transformers 🤖
* PyTorch ⚡

---

## 📦 Installation

Clone the repo:

```bash
git clone https://github.com/your-username/ai-translator-qwen2.git
cd ai-translator-qwen2
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000
```

---

## 🖥️ Features

* Translate English → Spanish
* Clean and minimal UI
* Runs fully offline
* No API keys required

---

## ⚠️ Limitations

* Small model → translations may not always be perfect
* First run downloads the model (may take a few minutes)
* Performance depends on your hardware
